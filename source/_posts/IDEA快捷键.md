| 快捷键            | 说明          |
| ----------------- | ------------- |
| `ctrl + alt + v`  | 快速声明对象  |
| `ctrl +  delete`  | 删除当前行    |
| `alt + enter`     | 异常处理      |
| `alt + ctrl + T`  | surround with |
| `shift + alt + ↑` | 当前行上移    |
| `ctrl + alt +L`   | 整理代码      |
| `ctrl + N`        | 构造函数      |
| `ctrl + I`        | 实现接口方法  |
| `ctrl+D`          | 复制上一行    |
| `ctrl+7`          | 查看类结构    |
| ctrl + J          | 水平分屏      |
| ctrl+ V           | 垂直分屏      |
